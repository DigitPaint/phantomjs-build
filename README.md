# Phantomjs-build

_Phantom js build for `ubuntu:14.04` created with Docker._

## Getting started

1. Pull this repository
2. `$ docker build .`
3. `$ docker images`
4. `$ docker run -t -i <image-id> /bin/bash`
5. `$ which phantomjs`
